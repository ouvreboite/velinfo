Velinfo'

Velinfo' est une application non officielle tentant d'identifier les stations bloquées avant le système interne de Velib'. Vous pouvez l'utiliser pour :

🗺️ Trouver des stations autour de vous via la carte.
❤️ Accéder directement à vos stations favorites.
🔎 Parcourir la liste des stations.

 A propos

Cette application est développée de manière amateur en se basant sur l'API Velib'. La détection des stations bloquées se base sur l'activité de chaque station sur le mois passé pour détecter les périodes d'inactivité anormales.

Les sources sont disponibles sur Github.
