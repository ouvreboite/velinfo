Velinfo'

Velinfo' est une application non officielle tentant d'identifier les stations Velib' bloquées au plus tôt. Vous pouvez l'utiliser pour :

  • Trouver des stations autour de vous via la carte.
  • Chercher directement dans la liste des stations.
  • Accéder directement à vos stations favorites.
Cette application est open-source. Les sources sont disponible sur Github.
